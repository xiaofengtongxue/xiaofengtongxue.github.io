首先准备工作：

安装CentOs，及其配置IP，DNS，通过Xshell连接虚拟机(开启ssh服务)，创建用户设置权限。
Ifconfig eth0 up/down ：开启/关闭 eth0网卡
Ifconfig eth0 192.168.188.110 （netmask 255.255.255.0）：设置eth0网卡的IP地址（此时设置的IP是临时的，重启消失）
Ifconfig：查看与设置IP地址，子网掩码
route add –net 192.168.188.0/24 gw 192.168.188.253 ：设置192.168.188.0网段的网关为192.168.188.253
route add –host 192.168.188.110 gw 192.168.188.253 ：设置192.168.188.110主机的网关为192.168.188.253
route add/del default gw 192.168.188.253 ：设置默认网关为192.168.188.253
 /etc/sysconfig/network-scripts/ifcfg-eth0/eth1 （此文件中添加ONBOOT=yes，不然每次重启网卡就会重写）
hostname：查看与设置主机名
 /etc/sysconfig/network
route：查看路由信息，默认网关
 /etc/resolv.conf ：设置DNS服务器
 service network start|restart|stop ：重启网卡
安装AMP及一些基础配置(管理权限，开机自启，防火墙配置等)，yum的操作(看看yum源)，安装vim
安装MySQL命令：yum –y install mysql-server
 开机自启：chkonfig mysql on
安装Apache命令：yum install httpd
 开机自启：chkconfig --levevs 235 httpd on
安装Php命令：yum –y inastall php
 目录：/var/wwww/
phpMyadmin目录：/usr/share/phpMyAdmin
Apache配置：/etc/httpd/conf.d/php.conf
