windows安全项检查

更新于 2016-08-02

天气这几天有点热，夜晚睡不着，工作效率也不好，几个插件一个星期了还没写好，下午坐在这突然想写python了，就看看os模块，分析了一下windows下安全注意项，不过Windows系统问题太多，要关注的地方也数不胜数，做日常检查，真的要一个命令一个命令的敲，关键都是在注册表，有点麻烦，这里主要用到系统自带的regini系统自带的注册表工具，用的还算舒服

  • 1、端口检查
  • 2、系统信息检查
  • 3、进程检查
  • 4、进程路径检查
  • 5、默认共享检查
  • 6、安全策略检查
  • 7、注册表启动项检查
  • 8、异常状态检查
  • 9、通信检查

完整代码如下：

#coding=utf-8
import os

if os.path.exists(str("D:/检查结果")):
    print("该文件夹存在")
else:
    os.mkdir(str("D:/检查结果"))
    print("创建成功")

print ("正在检查端口...")
os.system('netstat -anb >> D:\检查结果\端口信息.log')

print ("正在检查系统信息...")
os.system('systeminfo >> D:\检查结果\系统信息.log')

print ("正在检查进程...")
os.system('tasklist >> D:\检查结果\进程检查.log')

print ("进程路径检查...")
os.system('wmic process get name,executablepath,processid >> D:\检查结果\进程路径检查.log')

print ("默认共享检查...")
os.system('net share >> D:\检查结果\默认共享检查.log')

print ("安全策略检查...")
os.system('secedit /export /cfg LocalGroupPolicy&type LocalGroupPolicy >> D:\检查结果\安全策略检查.log')

print ("注册表启动项检查...")
os.system('reg query HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run & reg query HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run >> D:\检查结果\注册表启动项检查.log')

print ("异常状态检查...")
os.system('reg query HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows" "NT\CurrentVersion\SvcHost /s /v netsvcs&reg query HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows" "NT\CurrentVersion\SvcHost /s /v LocalService >> D:\检查结果\异常状态检查.log')

print ("检查结果保存于：D:\检查结果")
print ("小峰博客出品")
print ("http://www.lovexiaofeng.cn")
