00x3

1，打开虚拟机，用Xshell连接
2，按照实验要求进行命令输入：

/etc/passwd:

可以看出信息由6个分号组成的7个信息:
1:用户名
2:密码(已经加密)
3:UID(用户标识),操作系统自己用的
4:GID组标识
5:用户全名或本地帐号
6:开始目录
7:登录使用的Shell,就是对登录命令进行解析的工具。

/etc/shadow:

可以看出信息由8个分号组成的9个信息:
1，账号名称
2，密码（已加密）
3，最近更改密码的时间
4，密码不可被更改的天数
5，密码需要重新变更的天数
6，密码需要变更期限前的警告期限
7，密码过期的怒限时间
8，账号失效日期
9，保留

/etc/group:

可以看出信息由3个分号组成的4个信息:
1，用户组名称
2，用户组密码
3，GID
4，用户列表

分区的永久性挂载：

1，将虚拟机关机，在虚拟机设置上新增加一块2G硬盘，开机用Xshell链接
2，运行fdisk -l 查看分区情况，会发现多处以个2G的硬盘
3，对新硬盘进行分区：fdisk /dev/sdb  一步一步走下去
4，对新分区进行格式化操作：mkfs.ext3 /dev/sdb1
5，经刚分的区进行挂载在 /home/fans 目录下（我这里已经建立过fans目录）
mount -t ext3 -o rw /dev/sdb1 /home/fans
6，查看刚建分区的SSID：blkid /dev/sdb1
7，在/etc下vim fstab 将分区的信息添加在文件中，重启生效

虚拟机挂载硬盘：

 首先在虚拟机虚拟网卡上添加一块硬盘，然后重启虚拟机，运行fdisk –l 查看虚拟机主机硬盘的分区情况（如果之前是虚拟机安装的是一块硬盘，就会发现会多出一条信息，Disk: /dev/sdb 存在未分区的情况），然后就需要进行分区操作：
  fdisk /dev/sdb  机会让你选择对该硬盘进行操作 m（帮助） n（新建一块分区） d（删去硬盘上分区） w（将分区写在硬盘上 前提是已经将硬盘分好区了） 
  再次运行上面命令发现硬盘已经分区了，df –lh 看到硬盘并没有挂在到任何目录下，这是肯定的，因为还没有对硬盘进行挂载
  挂载命令：
    mkfs.ext3 /dev/sdb1  是对刚分好的区进行格式化
    mount –t ext3 –o rw /dev/sdb1 /home/fans  该命令是将/dev/sdb1硬盘挂载到/home/fans目录下（前提是在/home/下已经创建过fans目录，cd到该目录，命令mkdir 文件 进行创建）
ps 查看系统进程
  最好的参数 ps –aux （查看系统所有的进程）
top 以动态的方式进行动态的显示系统进程（跟win系统的任务管理器差不多的）
以上方式都可以显示进程的PID号
kill 终止进程
 kaiiall 终止由一个软件开启的多线程

第一节课：

安装CentOs，及其配置IP，DNS，通过Xshell连接虚拟机(开启ssh服务)，创建用户设置权限。
Ifconfig eth0 up/down ：开启/关闭 eth0网卡
Ifconfig eth0 192.168.188.110 （netmask 255.255.255.0）：设置eth0网卡的IP地址（此时设置的IP是临时的，重启消失）
Ifconfig：查看与设置IP地址，子网掩码
route add –net 192.168.188.0/24 gw 192.168.188.253 ：设置192.168.188.0网段的网关为192.168.188.253
route add –host 192.168.188.110 gw 192.168.188.253 ：设置192.168.188.110主机的网关为192.168.188.253
route add/del default gw 192.168.188.253 ：设置默认网关为192.168.188.253
 /etc/sysconfig/network-scripts/ifcfg-eth0/eth1 （此文件中添加ONBOOT=yes，不然每次重启网卡就会重写）
hostname：查看与设置主机名
 /etc/sysconfig/network
route：查看路由信息，默认网关
 /etc/resolv.conf ：设置DNS服务器
 service network start|restart|stop ：重启网卡
安装AMP及一些基础配置(管理权限，开机自启，防火墙配置等)，yum的操作(看看yum源)，安装vim
安装MySQL命令：yum –y install mysql-server
 开机自启：chkonfig mysql on
安装Apache命令：yum install httpd
 开机自启：chkconfig --levevs 235 httpd on
安装Php命令：yum –y inastall php
 目录：/var/wwww/
phpMyadmin目录：/usr/share/phpMyAdmin
Apache配置：/etc/httpd/conf.d/php.conf
